# Indo Vyapar App

React Native app (Expo) that loads the Indo Vyapar website in a WebView for both Android and iOS.

## URL loaded

`http://13.127.37.238:3004`

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start development server:

   ```bash
   npm run start
   ```

3. Run on Android:

   ```bash
   npm run android
   ```

4. Run on iOS:

   ```bash
   npm run ios
   ```
